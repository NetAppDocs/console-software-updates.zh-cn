= 使用软件更新启动ONTAP更新
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
确定ONTAP目标版本、修复并确认所有阻止程序后，您现在可以启动ONTAP存储系统的更新。

.步骤
. 从控制台左侧导航中，选择*健康*>*软件更新*。
+
此时将显示需要软件更新的群集列表。

. 确定您的存储系统的 ONTAP 目标版本。
+
有关识别 ONTAP 目标版本的信息，请参见 link:../ONTAP/choose-ontap-910-later.html["比较ONTAP版本以了解使用软件更新的优势和风险"]。

. 修复并确认 ONTAP 拦截器。
+
有关修复阻止程序的信息，请参见 link:../ONTAP/fix-blockers-warnings.html["在ONTAP升级之前使用软件更新修复所有阻止程序"]。

. 阅读并接受最终用户许可协议。
+
等待检索ONTAP映像。

. 单击“*安装更新*”以完成更新过程。
